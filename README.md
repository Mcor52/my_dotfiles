# my_dotfiles
Hyprland + Others (Arch dotfiles)

Just a dump of my Arch Hyprland dotfiles. I went for a Gruvbox-Material (Sainnhe) theme but with a focus on red.

I have scripts in the Dark/Light folders to switch between the themes easier. They remove the original config directories to make way for symlinks.

Make sure to chmod +x /path/to/script.sh if you'd like to run the scripts.
    -Important to note that these scripts assume a repo location of ~/Code/my_dotfiles
